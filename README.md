### `whoami`

I'm a full-stack developer based in Islamabad who's most valuable when the problem is complex and the solution needs to be clean. I architect the system, build the backend, and deliver the frontend — usually all on the same project. I've shipped software used across **300+ schools** and **17+ restaurant locations**, and I bring the same care to system design as I do to the last pixel on the screen.

- 🔭 Currently building **[AIO](https://aioapp.com)** — an AI-native, all-in-one restaurant operations platform (POS, inventory, staff, analytics)
- ⚙️ Deep in **NestJS microservices**, **AWS SQS** event-driven messaging, and real-time systems with **Ably**
- 🧱 Co-designing a **React 19 + React Compiler** frontend architecture as interim sub-lead
- 🤖 Driving an **AI-readiness transformation** of our codebase for the Claude Code ecosystem (agents, subagents, hooks, skills)
- 🎓 Electrical Engineering @ **NUST**, now writing TypeScript for a living
